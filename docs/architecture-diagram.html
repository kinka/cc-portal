⚙️ Core Services
ClaudeSessionManager
Session 管理器
  • • 生命周期管理
  • • 权限验证 (canAccessSession)
  • • 参与者管理
ClaudeSession
EventEmitter
  • • messages[] 历史记录
  • • pendingQueue 审批队列
  • • broadcast() 消息广播
ClaudeAgentBackend
Claude CLI 进程
  • • 子进程管理
  • • stdin/stdout 通信
  • • stream-json 协议
🗄️ Database (SQLite)
users
  • id (TEXT PRIMARY KEY)
  • max_sessions (INTEGER) DEFAULT 200
  • created_at (DATETIME)
sessions
  • id (TEXT PRIMARY KEY)
  • owner_id (TEXT) - FK to users
  • path (TEXT)
  • status (TEXT) - active/stopped/error
session_participants
  • session_id (TEXT) - FK
  • user_id (TEXT)
  • status (TEXT) - pending/joined
  • invited_at, joined_at (DATETIME)
📊 数据流
1
Alice 创建 Session
POST /sessions → 创建 session 记录 (owner_id=alice) → 返回 sessionId
2
Alice 邀请 Bob
POST /sessions/:id/invite {userId: "bob"} → 创建 session_participants (status=pending)
3
Bob 接受邀请
POST /sessions/:id/join → 更新 status=joined → Bob 成为参与者
4
发送消息
任意参与者 POST /messages → Claude 处理 → SSE 广播给所有参与者
5
实时接收响应
所有参与者通过 GET /stream (SSE) 实时接收 Claude 的流式响应
